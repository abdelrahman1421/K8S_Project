# App deployment 
Deploying webapp with backend MongoDb.

## How to deploy it
Make sure you have a Kubernetes cluster
```
git clone
kubectl apply -f ./MongoDB_Mongo_Express_Deployment/.
kubectl apply -f ./Webapp_Deployment.

```
